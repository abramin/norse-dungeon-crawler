# Norse Dungeon Crawler

A browser-based roguelike prototype built with React, Vite, and Tailwind CSS. Explore an icy dungeon inspired by Norse mythology, battle enemies, solve riddles, and collect loot on your way to the Frost Giant boss.

## Getting Started

### Prerequisites
- Node.js 18 or newer
- npm (bundled with Node.js)

### Installation
```bash
npm install
```

### Development server
Start Vite's development server with hot module replacement:
```bash
npm run dev
```
Then open the printed local URL in your browser (default: http://localhost:5173).

### Production build
Create an optimized production build:
```bash
npm run build
```

Preview the built app locally:
```bash
npm run preview
```

## Project Structure
- `src/App.tsx`: Main game component containing dungeon logic, combat, riddles, and UI.
- `src/main.tsx`: React entry point that mounts the app.
- `src/index.css`: Tailwind directives and global styling.
- `index.html`: Root HTML template used by Vite.
